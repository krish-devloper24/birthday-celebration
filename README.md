# 🎂 Premium Birthday Celebration Website

A beautiful and interactive Birthday Celebration Website built using HTML, CSS, and JavaScript.

## ✨ Features

- 🎉 Personalized Birthday Wishes
- 🎂 3D Birthday Cake Animation
- 🎈 Floating Balloons
- ✨ Neon Glow Effects
- 🎆 Fireworks Animation
- 📱 Responsive Design
- 🎁 Custom Name Input

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Download or Clone the Repository.
2. Open `index.html` in your browser.
3. Enter a name and click **Celebrate 🎉**.
